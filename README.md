# YOLOv5 Fine-Grained Object Detection for Aerial Images

## Overview

This project implements a Fine-Grained Object Detection System for Aerial Images using YOLOv5. The system can detect and classify objects from aerial imagery and provide contextual information through a voice-assisted interface.

## Features

- Real-time object detection
- Fine-grained object classification
- Voice-based object information
- User authentication system
- Interactive web application
- High detection accuracy using YOLOv5

## Technologies Used

- Python
- YOLOv5
- OpenCV
- PyTorch
- Streamlit
- SQLite

## Installation

1. Clone the repository

```bash
git clone https://github.com/Anjum05-git/YoloV5-Fine-grained-object-detection.git
```

2. Navigate to the project folder

```bash
cd YoloV5-Fine-grained-object-detection
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
streamlit run app.py
```
